Pycoast
=======

Pycoast is a Python package to add coastlines, borders, and rivers to raster
images using data from the GSHHS and WDBII datasets. It also has functionality
for adding logos, text, and colorbars (scales).

.. versionadded:: 1.2

   Adding logos, text, and colorbars was added by absorbing the now deprecated
   pydecorate package.
